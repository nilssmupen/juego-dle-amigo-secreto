# 🎁 Sorteador de Amigo Secreto

Este es un pequeño proyecto en **JavaScript**, **HTML** y **CSS** que permite agregar nombres de amigos a una lista y sortear aleatoriamente uno de ellos como "Amigo Secreto".  
Incluye validaciones para evitar nombres repetidos o entradas vacías y limpia la lista después del sorteo.

## 🚀 Características
- Agregar nombres a la lista.
- Validar que no se ingresen nombres vacíos o duplicados.
- Mostrar la lista de amigos agregados.
- Sorteo aleatorio de un amigo secreto.
- Limpieza automática de la lista después del sorteo.

## 📌 Uso
1. **Clonar el repositorio**
   https://github.com/nilssmupen/juego-dle-amigo-secreto.git
   
2. Abrir el proyecto

3. Localiza el archivo index.html.

4. Ábrelo en tu navegador.

5. Agregar amigos

6. Escribe un nombre en el campo de texto.

7. Pulsa "Añadir" para guardarlo en la lista.

8. Sortear

9. Pulsa "Sortear" para elegir aleatoriamente un amigo secreto.

10. La lista se limpiará y solo se mostrará el resultado.
